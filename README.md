
# Backlash Politics and LLM-Assisted Political Text Classification

This repository contains a mixed-methods political science research project examining backlash-oriented political discourse in European populist and radical-right parties between 2018 and 2025.

The project combines:

* comparative manifesto analysis,
* manual qualitative coding,
* and exploratory Large Language Model (LLM)-assisted text classification.

Using manifesto excerpts collected from European populist-party programs and manifestos, the project explores whether contemporary LLMs can assist thematic classification of political texts associated with nationalism, anti-immigration discourse, traditional values, law and order, and anti-elite populism.

The project was developed using R, Quarto, and comparative qualitative coding methods.

---

# Research Question

Can Large Language Models assist qualitative coding of backlash-oriented political discourse in European populist-party manifestos?

---

# Project Overview

The project consists of three main components:

## 1. Comparative Manifesto Analysis

Using data from the Manifesto Project Database (MPD), the project analyzes rhetorical trends across European populist and radical-right parties between 2018 and 2025.

The analysis focuses on:

* nationalism,
* traditional morality,
* anti-multiculturalism,
* and law-and-order discourse.

Descriptive statistics and comparative visualizations were produced in R.

---

## 2. Comparative Qualitative Corpus

A multilingual corpus of political manifesto excerpts was manually constructed from party programs, electoral manifestos, and government programmes associated with:

* Vox (Spain)
* Alternative für Deutschland (Germany)
* Rassemblement National (France)
* Fratelli d’Italia (Italy)
* Freedom Party of Austria — FPÖ (Austria)
* Sweden Democrats (Sweden)

The corpus includes:

* original-language excerpts,
* English translations,
* thematic coding categories,
* document metadata,
* and manually assigned ideological classifications.

---

## 3. LLM-Assisted Classification

The project compares manually coded classifications with outputs generated by:

* ChatGPT
* Qwen (open-source LLM)

Both models received the same zero-shot classification prompt and classified each excerpt into one of five thematic categories:

* Nationalism_Sovereignty
* Anti_Immigration_Multiculturalism
* Traditional_Values
* Law_Order_Security
* Anti_Elite_Populism

The project then compares agreement levels between manual and LLM-generated classifications.

---

# Key Findings

| Model   | Agreement Rate |
| ------- | -------------- |
| ChatGPT | 87.9%          |
| Qwen    | 85.5%          |

The findings suggest that contemporary LLMs can identify broad ideological patterns in political texts with relatively high consistency, although disagreement cases reveal important conceptual overlaps between nationalism, populism, immigration discourse, and security rhetoric.

# Coding Framework

Each manifesto excerpt was manually classified into one primary thematic category:

| Category                          | Description                                                    |
| --------------------------------- | -------------------------------------------------------------- |
| Nationalism_Sovereignty           | National identity, sovereignty, patriotism, anti-EU governance |
| Anti_Immigration_Multiculturalism | Immigration restriction, border control, anti-multiculturalism |
| Traditional_Values                | Family values, Christianity, traditional morality              |
| Law_Order_Security                | Crime, policing, terrorism, public order                       |
| Anti_Elite_Populism               | Elite-versus-people framing, anti-establishment rhetoric       |
